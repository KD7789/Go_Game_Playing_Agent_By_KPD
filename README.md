# Go_Game_Playing_Agent_By_KPD
A Go-playing agent implemented in Python using Minimax search with Alpha-Beta Pruning for efficient game tree exploration.

This project implements an AI agent for playing Go as part of the Foundations of AI (FAI) course. The agent uses Minimax search to evaluate game states and select optimal moves, enhanced with Alpha-Beta Pruning to significantly reduce the search space by eliminating branches that cannot influence the final decision.

Key components include a Go game engine with rule enforcement (captures, Ko rule, scoring), a heuristic evaluation function for board state assessment, and a configurable search depth to balance performance and move quality.
